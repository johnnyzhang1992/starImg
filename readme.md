## About StarImg

StarImg 是一个明星图库，主要采集微博和Ins的明星图片，后期用户也可也自行上传。

## 前端页面

 [starimg.cn](https://starimg.cn) 网站的前端页面。
 laravel框架，php + react.js


## 后台管理页面
