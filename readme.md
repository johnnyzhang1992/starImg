## About StarImg

StarImg 是一个明星图库，主要采集微博和Ins的明星图片，后期用户也可也自行上传。

## notice

微博来源的图片显示403，无权限访问，貌似是被微博拉黑了。。。

所以暂时去ins部分改为默认。

## 前端页面

 [starimg.cn](https://starimg.cn) 网站的前端页面。
 laravel框架，php + react.js
 
 使用了 pinterest开源的项目 [gestalt](https://github.com/pinterest/gestalt)
